

# Bible Home Page — Your Personal Browser Start Page

## Overview
A warm, peaceful browser start page with earth tones, serif fonts, and a calming aesthetic — designed for daily spiritual reflection and writing project management.

## Layout
A single-page dashboard with a soft parchment-like background, organized into clean sections:

### 1. Bible Verse of the Day
- Fetches a daily verse from a free Bible API (e.g., bible-api.com)
- Displays the verse prominently at the top with scripture reference
- Option to generate a shareable image card with the verse overlaid on a serene background/pattern
- Download button for the verse image

### 2. Writing Projects Tracker
- Editable list of Bible writing projects stored in browser localStorage
- Each project has a title, description, and status (e.g., In Progress, Completed, On Hold)
- Add, edit, and delete projects directly from the UI
- Simple progress indicators

### 3. Daily Diary / Notes
- Multiple timestamped entries per day
- Browse past days using a date picker/calendar
- Entries saved in browser localStorage
- Quick "Add Note" button with a text area
- Entries shown in reverse chronological order

## Design
- Warm earth-tone palette (cream background, muted browns, soft gold accents)
- Serif headings, clean sans-serif body text
- Subtle decorative elements (thin dividers, gentle shadows)
- Responsive but optimized for desktop use as a home page

## Data Storage
- All data (projects, diary entries) persisted in browser localStorage — no backend needed, fully private and local
